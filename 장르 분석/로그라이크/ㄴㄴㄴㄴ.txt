4요소에서 두가지를 빼고 추가 설명하는 이유 없음, 할꺼면 같이 하던가

로그-라이크 단어는 빼도 괜찮음 

식별 요소, 자원 관리, 영원한 죽음, 랜덤 생성 간의 상호 작용

영원한 죽음 -> 랜덤 생성 : 반복 플레이의 이유를 만듬
랜덤 생성 -> 영원한 죽음 : 반복 플레이시 발생하는 지루함을 상쇄

영원한 죽음 -> 식별 요소 : 앞으로 일어날 일을 예상 또는 관찰하게됨
식별 요소 -> 영원한 죽음 : 선택을 무를 수 없다는 점에서 긴장감과 함께 신중한 선택을 하게 됨

영원한 죽음 -> 자원 관리 : 유저의 기본 자원을 관리해야 하는 이유를 만들어줌
자원 관리 -> 영원한 죽음 : 던전에서 반복적인 사냥으로 일명 파밍을 하지 못하도록 함

1. 영원한 죽음-> 랜덤 생성
랜덤 생성 -> 영원한 죽음

2. 영원한 죽음 -> 자원 관리
자원 관리 -> 영원한 죽음

3. 영원한 죽음 -> 식별 요소
식별 요소 -> 영원한 죽음

4. 랜덤 생성 -> 식별 요소
식별 요소 -> 랜덤 생성

5. 랜덤 생성 -> 자원 관리
자원 관리 -> 랜덤 생성

6. 식별 요소 -> 자원 관리
자원 관리 -> 식별 요소
